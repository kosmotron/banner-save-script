# 📄 Инструкция по работе с шаблоном баннера

## 🎯 Для чего нужен скрипт

Скрипт автоматизирует рутинные действия при создании баннера:
- **Экономит время** — не нужно вручную сохранять PSD и экспортировать JPEG
- **Исключает ошибки** — имя файла всегда формируется правильно из текстовых слоёв
- **Стандартизирует** — все баннеры сохраняются по единому шаблону именования

---

## 🚀 Установка (делается 1 раз)

1. Скопируйте `SavePSD_and_ExportJPEG.jsx` из папки **«Технические файлы»** в папку Photoshop:
   - **Windows**: `C:\Program Files\Adobe\Adobe Photoshop [версия]\Presets\Scripts\`
   - **macOS**: `/Applications/Adobe Photoshop [версия]/Presets/Scripts/`
2. **Перезапустите** Photoshop

---

## 📝 Как пользоваться (каждый раз)

1. **Впишите** данные в слои:
   - `Название мероприятия` → например, `Фестиваль дружбы`
   - `Дата мероприятия` → например, `15-16 августа 2026`
2. **Запустите скрипт**: `Файл → Сценарии → SavePSD_and_ExportJPEG`
3. **Выберите папку** для сохранения
4. **Готово!** PSD и JPEG сохранены с именем: `НАЗВАНИЕ 15-16 августа 2026`

---

## ⚡ Настройка горячей клавиши

1. `Окно → Действия` → **«Создать новое действие»**
2. Назначьте клавишу (например, **F2**)
3. Нажмите **«Записать»** → выполните скрипт → нажмите **«Стоп»**

Теперь просто нажимайте **F2** после ввода данных!

---

## 📂 Как изменить папку сохранения

По умолчанию скрипт **предлагает выбрать папку вручную** при каждом запуске. Если хотите изменить это поведение:

### Вариант 1: Использовать постоянную папку
Замените в скрипте этот блок:
```javascript
var folder = Folder.selectDialog("Выберите папку для сохранения баннера");
```
На:
```javascript
var folder = new Folder("O:/Ваш/Путь/к/папке/Баннеры");
```

### Вариант 2: Сохранять рядом с PSD
```javascript
var folder = new Folder(doc.path);
```

### Вариант 3: Сохранять в папку на уровень выше
```javascript
var folder = new Folder(doc.path + "/..");
```

## ⚠️ Важно

- **Не переименовывайте** слои `Название мероприятия` и `Дата мероприятия`
- Скрипт автоматически удаляет запрещённые символы: `\ / : * ? " < > |`
- Название переводится в **ЗАГЛАВНЫЙ РЕГИСТР**, дата остаётся как есть
- Качество JPEG: **10 из 12** (отличное)
